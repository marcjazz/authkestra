# authly-token

JWT and token utilities for [authly-rs](https://github.com/marcjazz/authly-rs).

This crate provides JWT signing, verification, and token abstraction for use within the `authly` framework.

## Features

- JWT signing and verification.
- Token validation helpers.
- Integrated with `chrono` for expiration handling.

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
authly-token = "0.1.0"
```

## Part of authly-rs

This crate is part of the [authly-rs](https://github.com/marcjazz/authly-rs) workspace.
