# Authkestra Community Onboarding

Welcome to the Authkestra community! This guide will help you get started as a contributor, understand the project's vision, and find resources to learn and collaborate effectively.

---

## 🧭 North Star

**Authkestra = composable, verifiable auth primitives for humans and AI systems.**

---

## 🚀 Getting Started

1. **Read the Vision**
   - See [docs/README.md](./README.md) and [conversion.md](../conversion.md) for the big picture.
2. **Understand the Roadmap**
   - Review [docs/roadmap.md](./roadmap.md) for current and future milestones.
3. **Pick an Area**
   - Engine, providers, adapters, admin UI, SDKs, etc.
4. **Join the Conversation**
   - [GitHub Discussions](https://github.com/marcjazz/authkestra/discussions)
   - Propose RFCs for major changes
5. **Start Contributing**
   - Look for issues labeled `good first issue` or `help wanted`
   - Ask questions in Discussions or open a draft PR
